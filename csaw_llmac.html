This page hosts the leaderboard for the CSAW'24 LLM CTF Attack Challenge being held from 28th October to 8th November 2024. The challenge asks participants to create an LLM agent that can solve CTF challenges. Examples of LLM agents are the baseline agent presented in the NYU CTF Dataset, and the EnIGMA agent that achieves state-of-the-art results.
